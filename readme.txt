# 概要
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

指定した時間にアラームを鳴らすか、ソフトを起動させるソフトです。

AutoHotkey で作られています。

＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
# 主な特徴
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

「時刻で指定」する場合と
「今からの時間で指定」する場合
の２種類の指定方法があります。


時間を指定してスタートボタンを押すと、
その時間にアラームを鳴らします。


ただし、正確な時間を図るのには向いていません。
指定された時間を大雑把にしか認識しないところがあります。


＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
# 動作環境
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

AutoHotkey上で動作します。  
AutoHotkey_L  Unicode版  の事をいいます。

OS は、Windowsでのみ使用できます。  
Windows XP / 2003 / Vista / 7 / 8 / 10 (32-bit、64-bit両対応)

AutoHotkey_L が動作する環境であれば可能だと思います。

-----------------------------------------------------

※ AutoHotkey の派生版である 「AutoHotkey_L Unicode版」でのみ動作します。

元の AutoHotkey では使えません。


それと  AutoHotkey 1.1.xx のバージョンのみで使えます。  
AutoHotkey 2.xx のバージョンでは使えません。


＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
# ファイルの構成
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

ListTimerBU.exe				本体です。  

doc（フォルダ）				音声用のファイルが入っています。
							アイコンも入っています。

	doc\backGround			決められたファイル名の画像を置くと  
							ソフトのウインドウ内に反映されます。

user（フォルダ）			ユーザーが設定した事はこのフォルダ内に反映されます。


userフォルダの中身がカラの場合、
ListTimerBU.exe を実行すると、自動的に作成するようになっています。


＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
# インストール
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

現時点では、インストーラーはありません。

○レジストリはいじりません  
　USBなどのストレージから使用することも可能です。


ダウンロードしたファイルを解凍ソフトで展開してください。

その後に好きなフォルダ内に配置して使ってください。  
ListTimerBU.exe  を実行すると、  
このソフトに必要なファイルやフォルダが同じフォルダ内に自動的に作成されます。

----------------------------------------------------

Program Files  などの UAC管理下のフォルダの中では  
このソフトは使用できません。

それ以外のフォルダに配置して使ってください。

----------------------------------------------------

# バージョンアップ
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

1. ListTimerBU を必ず終了させておいてください。

2. 解凍したばかりならばそのままフォルダ内のファイルを
   すべて上書きすればOKです。
-------------------------------------------------
##　「user」のフォルダをバックアップしておく事を薦めます。
-------------------------------------------------

　もし、ListTimerBU を実行したまま上書きしてしまった場合、
　設定用のファイルなどが、作者が想定しない形で上書きなどをするかもしれません。

　それによるトラブルが発生するかもしれないので、
　バージョンアップする際は、念のため  
　「user」  
　のフォルダをバックアップしておく事を薦めます。


＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
# バックアップ
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

	「user」のフォルダをどこかに保存しておいてください。

	ユーザーが設定した内容はそのフォルダ内のファイルに設定内容が保存されます。
